# joao.codes

Personal profile site for [joao.codes](https://joao.codes), built with Vite and React.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## Deploy

Vercel settings:

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
